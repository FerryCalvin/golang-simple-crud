# Golang Simple CRUD

A simple **CRUD (Create, Read, Update, Delete)** REST API built with **Golang**.  

---

## ✨ Features
- ➕ Create new records  
- 📖 Read records (all or by ID)  
- ✏️ Update records  
- ❌ Delete records  

---

## 🛠 Tech Stack
- **Language**: Go (Golang)  
- **Framework/Router**: net/http 
- **Database**: MySQL
